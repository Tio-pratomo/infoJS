## Skenario Project

Anda adalah Backend Engineer di sebuah startup e-commerce. Tim Product meminta fitur rekomendasi otomatis. Jika User A melihat "Tas Laptop", sistem harus menyarankan barang lain yang sering dibeli bersamaan dengan tas tersebut oleh user lain (misal: Mouse, Keyboard).

**Masalah:**
Menggunakan `if-else` manual sangat tidak scalable. Kita butuh struktur data yang bisa mempelajari relasi antar produk secara otomatis.

**Solusi:**
Gunakan **Graph**.

- **Vertex (Node)** = Produk.
- **Edge (Garis)** = Hubungan pembelian bersamaan.
- **Weight (Bobot)** = Seberapa sering dibeli bersamaan. Semakin tebal garisnya, semakin kuat rekomendasinya.

## Langkah 1: Membuat Core Graph (`core/graph.js`)

Ini adalah "otak" dari sistem kita. Graph ini mencatat produk apa berteman dengan produk apa.

```javascript
class Graph {
  constructor() {
    this.adjacencyList = {}; // Hash Table untuk simpan node & tetangganya
  }

  addVertex(product) {
    if (!this.adjacencyList[product]) {
      this.adjacencyList[product] = [];
    }
  }

  addEdge(product1, product2) {
    // Cari apakah hubungan sudah ada?
    const existingEdge = this.adjacencyList[product1].find((edge) => edge.node === product2);

    if (existingEdge) {
      // Jika sudah ada, perkuat hubungannya (tambah bobot)
      existingEdge.weight += 1;
    } else {
      // Jika belum, buat hubungan baru dengan bobot awal 1
      this.adjacencyList[product1].push({ node: product2, weight: 1 });
    }

    // Lakukan hal yang sama untuk product2 (Undirected Graph)
    // ... (kode cermin untuk product2)
  }

  // Mengambil rekomendasi (tetangga dengan bobot tertinggi)
  getRecommendations(product) {
    const neighbors = this.adjacencyList[product] || [];
    // Urutkan berdasarkan weight (terbesar ke terkecil)
    return neighbors.sort((a, b) => b.weight - a.weight);
  }
}
```

## Langkah 3: Logic Rekomendasi (`services/recommendation.js`)

Di sini kita "melatih" Graph kita menggunakan data transaksi.

```javascript
const graph = new Graph();

// 1. Feed Data ke Graph
PURCHASE_HISTORY.forEach((transaction) => {
  // Setiap barang dalam satu transaksi dianggap saling berhubungan
  for (let i = 0; i < transaction.length; i++) {
    graph.addVertex(transaction[i]);

    for (let j = i + 1; j < transaction.length; j++) {
      // Hubungkan setiap produk dengan produk lainnya dalam keranjang yang sama
      graph.addEdge(transaction[i], transaction[j]);
    }
  }
});

// 2. Test Rekomendasi
const productSedangDilihat = 'Headset Gaming';
const hasil = graph.getRecommendations(productSedangDilihat);

console.log(`Karena Anda melihat "${productSedangDilihat}", kami sarankan:`);
hasil.forEach((item) => {
  console.log(`- ${item.node} (Skor Relevansi: ${item.weight})`);
});
```
